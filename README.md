# NG-BVN-Verifier
A Python/Django app for Nigerian BVN Verification

It uses the Flutterwave python bindings for BVN verification.

Dependencies:
<p>
<ol>
    <li>Django==1.10</li>
    <li>Flutterwave==1.0.3</li>
    <li>Naked==0.1.31</li>
    <li>pycrypto==2.6.1</li>
    <li>PyYAML==3.12</li>
    <li>requests==2.11.1</li>
    <li>shellescape==3.4.1</li>
    <li>wheel==0.24.0</li>
</ol>
<p>

<p>
